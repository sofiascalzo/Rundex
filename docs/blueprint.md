# **App Name**: Rundex

## Core Features:

- User Authentication: Secure user login and registration with email and password.
- Bluetooth Connection: Enable connection to BLE transmitters to receive motion data via the Web Bluetooth API.
- Real-time Data Display: Display real-time running data such as speed, acceleration, and posture errors in an interactive dashboard.
- Data Visualization: Present running data through dynamic graphs and charts using Recharts, highlighting performance metrics and posture analysis.
- Posture Error Detection: Analyze sensor data to detect and visualize posture errors during the run, providing insights for improvement.
- File Upload for Mock Data: Allow users to upload .json or .csv files containing run data to simulate sensor input for testing and demonstration purposes.
- Predictive Performance Tool: Use generative AI to provide performance predictions based on running data, such as estimated finish times and optimal pacing strategies, that incorporate environmental factors.

## Style Guidelines:

- Primary color: Vibrant orange (#F97316) to convey energy and motivation.
- Background color: Dark grey (#1E1E1E) to provide a sleek, modern, sporty feel.
- Accent color: Lime green (#A7F35D) for highlights and interactive elements, creating contrast.
- Headline font: 'Orbitron', sans-serif, for a futuristic, tech-inspired feel.
- Body font: 'Inter', sans-serif, for a clean and readable interface.
- Use modern, minimalist icons to represent different data metrics and functionalities.
- Implement a responsive layout optimized for both desktop and mobile devices.
- Incorporate smooth, subtle animations using framer-motion to enhance user experience and data loading.
